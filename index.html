WebAR Examples!

아래 링크에 접속해서 카메라가 켜지면, 위 이미지를 비춰보세요(From HiuKim Yuen's course)

  • Example 1 - Simple Image Tracking
    기본적인 Image Tracking
  • Example 2 - Camera Mock
    카메라로 테스트하기 번거로운 경우에 이미지를 보여주는 영상을 대체하여 테스트
  • Example 3 - Custom Container
    전체화면이 아니라 축소(custom contaniner)하여 테스트
  • Example 4 - 3D Model Basic
    3D 모델 로드하기(기초)
  • Example 5 - Multiple Image Tracking
    Multipl Image Tracking: 동시에 tracking 되는 것은 아니고, 각각 하나의 target만 tracking
  • Example 6 - Simultaneous Multipl Image Tracking
    이미지에 연결된 별개의 target을 동시에 tracking
    Example 5와 6에서 사용한 이미지: 1번, 2번
  • Example 7 - 3D Model Animation
    3D 모델 애니메이션
  • Example 8 - Events Handling
    Target Detection 성공 여부를 구분하는 기능, console log로 확인할 수 있음
  • Example 9 - Audio Contents
    오브젝트와 함께 배경음악이 재생됨, 공간음향이 지원되어 오브젝트에 가까이 가면 소리가 커짐
  • Example 10 - User Interaction
    오브젝트를 클릭 또는 터치하면 다른 소리가 재생됨
  • Example 11 - Video Player
    비디오가 plane의 texture 역할을 함
  • Example 12 - Chroma Video
    그린 스크린에서 제작한 영상의 배경을 없애고 영상 재생
  • Example 13 - CSS Renderer
    CSS로 간단한 오브젝트 추가하기
  • Example 14 - Youtube Streaming
    Vimeo 플랫폼은 iframe 태그로 처리해도 증강된 비디오를 손으로 조작할 수 있어 간단하지만, Youtube 영상은 별도의 API가 필요함.
